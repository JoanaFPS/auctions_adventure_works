# Adventure Works Auctions

This was a group project whose main goal was to extend the AdventureWorks database with an Auction schema to support an online stock clearance auction campaign. The solution was implemented in T-SQL through an idempotent script that creates the required tables, configuration settings, indexes, and stored procedures for adding products to auctions, placing bids, removing auctioned products, updating auction statuses, and viewing customer bid history.

This project uses the AdventureWorks sample database.
